# Extended Land Use Classification and Analysis using Sentinel-2 Imagery

## Overview
This repository extends my original Final Year Project on deep learning-based land use classification using Sentinel-2 multispectral imagery. While the original project focused on accurately predicting multiple land cover classes using Convolutional Neural Networks (CNNs), this extension builds upon that foundation by introducing **semantic interpretation** and **temporal analysis capabilities**.

The goal of this extension is to move beyond raw classification outputs and provide a more meaningful understanding of the environment by linking predicted land cover classes to **real-world vegetation types, ecosystems, and landscape characteristics**, as well as enabling the analysis of how these landscapes **change over time**.

## What’s New in This Extension

### Semantic Land Cover Interpretation
- Introduces a **mapping layer** between predicted land cover classes and:
  - Vegetation types (e.g. forests, shrublands, crops)
  - Typical plant species (e.g. olive trees, pine, grasses)
  - Ecosystem characteristics
- Enhances predictions with **human-readable descriptions**
- Bridges the gap between **model output and real-world understanding**

---

### Temporal Analysis of Land Use
- Enables analysis of **land cover changes over time**
- Supports comparison of satellite imagery across different time periods
- Allows identification of:
  - Urban expansion
  - Deforestation or vegetation growth
  - Changes in agricultural patterns
- Lays the foundation for **environmental monitoring and trend analysis**

---

### Enhanced Interpretability
- Extends model outputs with contextual insights
- Improves usability for:
  - Environmental monitoring
  - Urban planning
  - Research and analysis
- Moves towards a more **explainable AI system**

---

## Why This Matters
Traditional land cover classification systems provide categorical outputs (e.g. "forest", "urban", "pasture"). While useful, these labels lack contextual meaning for deeper analysis.

This extension enhances the system by:
- Connecting predictions to **ecological and environmental context**
- Providing **interpretable insights** instead of raw labels
- Enabling **temporal tracking of land use changes**

This makes the system more valuable for real-world applications where understanding **how and why landscapes change** is just as important as identifying them.
